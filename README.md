# My-rocket-fly-computer
this is the code and software i use for my model rocket to control and log data from the flight 

## Goal

- log fly data
- 2 pyro chanel
- quite light
- Arduino C++ firmware

## Parts
- computer : Teensy 3.6 dev board
- gyroscope : mpu6050
- altimetre : bosch bmp280
- n chanel mosfet for pyro
- tricolor Led
