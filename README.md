# Venona flight computer
the goal of this flight computer is to be easy to manufacture be mobulable for all kind of mission 

## Goal

- log fly data
- 4 pyro chanel
- quite light
- Arduino C++ firmware

## Parts
- Teensy 3.6 dev board
- mpu6050
- bosch bmp280
- n chanel mosfet for pyro
- tricolor Led
- buzzer
